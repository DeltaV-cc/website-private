# Delta V brand kit

Signal-led assets for Delta V: dark systems, restrained cyan, and a human layer of orchid/purple.

## Folder map

- `logo/delta-v-logo.png` — current favicon/logo raster, 64 × 64.
- `logo/delta-v-logo.svg` — scalable geometric mark for print and product surfaces.
- `logo/delta-v-color-palette.svg` — palette reference.
- `ink-garden/ink-garden-panorama.webp` — production web background.
- `ink-garden/ink-garden-panorama.png` — lossless panorama.
- `ink-garden/ink-garden-source.png` — original source export.
- `ink-garden/ink-garden-source.jpg` — compact source export.
- `ink-garden/lotusbloom.gif` — animated Ink Garden variant.
- `banner/delta-v-ink-garden-banner.svg` — ready-to-use 1600 × 760 banner.
- `brand.css` — portable tokens and banner styles.

## Usage

The files are served under `/website-private/brand/` in the static site. Keep the mark on dark backgrounds, use cyan for primary signals, and reserve purple/orchid for secondary or Forge moments. Use the WEBP panorama for web delivery; keep PNG/JPG for handoff and editing.
