# IntelHub Audit & Improvement Plan

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Audit all 4 IntelHub dashboards (Macro, AI, Web3, Infosec), identify missing data, UX gaps, and propose concrete improvements.

**Architecture:** IntelHub is a client-side fetched dashboard app — 4 tabs, 17 components, 2103 lines. Data comes from 10+ APIs via hooks.ts (566 lines). CORS-blocked APIs go through a Cloudflare Worker proxy.

**Tech Stack:** Next.js 16 (static export), TypeScript, Tailwind CSS, DeFiLlama/Polymarket/CISA/NVD/HIBP APIs.
